# TvRageWrapper

A wrapper gem for the TVRage API. *Not ready for use yet!*

## Installation

Add this line to your application's Gemfile:

    gem 'tv_rage_wrapper'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tv_rage_wrapper
